# find
## find -iname
 ```
[cs15lwi23agm@ieng6-202]:written_2:529$ find -iname ch1.txt
./non-fiction/OUP/Abernathy/ch1.txt
./non-fiction/OUP/Berk/ch1.txt
./non-fiction/OUP/Fletcher/ch1.txt
./non-fiction/OUP/Kauffman/ch1.txt
./non-fiction/OUP/Rybczynski/ch1.txt
```
We can find the path of the file ```ch1.txt``` by using this command and find out the path of five ```ch1.txt``` in written_2.
```
[cs15lwi23agm@ieng6-202]:written_2:530$ find -iname boston-whereToGo.txt
./travel_guides/berlitz2/Boston-WhereToGo.txt
```
We can check where is the file by the name ```boston-whereToGo.txt``` here and it will return the path even if it does not exactly same as ```boston-whereToGo.txt```, which let us find out ```Boston-WhereToGo.txt``` here. If the file can be found, it will return the path of it. It is useful when we want to check where are the files with the file name.
